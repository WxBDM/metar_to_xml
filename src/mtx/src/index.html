METAR Parser

Step 1:

Copy a METAR from the Aviation Weather Center.

Step 2:

Go to terminal, navigate to where xml_maker is located and type: 

            python3 xml_maker.py

and paste the METAR from the clipboard and hit enter.

Step 3:

Click this button to display results:

Parsed METAR

Follow the steps above. If done correctly, this section will display a table.

You can download the associated XML file here.
